Topic Focus: Solving Quadratic Equations

Quadratic equations are equations of the form ax² + bx + c = 0. They represent parabolas and can be solved in several ways. Let’s go through the most common method — factoring.

Example Problem:

Solve: x² + 5x + 6 = 0

Step-by-Step Solution:

  1. Identify a, b, and c: a = 1, b = 5, c = 6.
  2. Find two numbers that multiply to c (6) and add up to b (5): Those numbers are 2 and 3.
  3. Write factors: (x + 2)(x + 3) = 0
  4. Set each factor equal to zero: x + 2 = 0 → x = -2 x + 3 = 0 → x = -3
  5. Check your answers: Substitute x = -2 and x = -3 into the original equation to verify both work.

Final Answer: x = -2, -3

Why Algebra Matters

Algebra strengthens your logical thinking and helps you analyze patterns. From coding and finance to architecture and science, algebraic thinking is used everywhere to solve problems and make predictions.

Study Tips

  • Work on small problems daily to build fluency.
  • Write each step clearly when solving equations.
  • Check your solutions by substitution.
  • Use online graphing tools to visualize functions.
